# Online Sales Analysis

# Monthly Revenue & Order Analysis
# Task Overview

This task analyzes the monthly revenue and order count for a business using SQL. The goal is to understand sales trends, identify top-performing months, and present insights in a structured way.

# The analysis covers:

Total revenue per month

Number of unique orders per month

Top-performing months by revenue

Filtered data for a specific year

# Tools & Technologies

Database: SQL Server

SQL Functions Used:

SUM() – to calculate total revenue

COUNT(DISTINCT order_id) – to count unique orders

GROUP BY – to aggregate monthly data

ORDER BY – to sort results

WHERE – to filter by year

TO_CHAR()(SQL Server) – to format month-year


# Data Cleaning & Preparation

Ensured all order data is correct and complete

Removed duplicates if any

Filtered data for the required year (e.g., 2024)

Aggregated data monthly for better readability


# Key Learnings

Learned how to filter, sort, and limit data for insights

Improved skills in data formatting for better readability

# Project Outcome
